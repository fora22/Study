# Rule
- 매주마다 계획 Issue를 올린다.
- 다른 레포지토리 에 있는 계획이면 링크를 단다.
- Issue 이름은 몇년도 몇주차인지를 적어 놓는다.
  - 구글 캘린더 기준으로 작성함
- 작업 시작, 종료 comment를 남겨 시간을 기록한다.
  - 작업에 해당하는 레포지토리도 상관 없음
